# gulp配置
## gulp相关插件
|  插件   | 描述  |
|  ----  | ----  |
| gulp-less  | 编译less |
| gulp-clean-css  | 压缩css |
| gulp-babel  | 编译ES6 |
| babel-polyfill  | 编译ES6新的API（生产依赖） |
| gulp-uglify  | 压缩js |
| gulp-imagemin  | 压缩图片 |
| imagemin-pngquant  | 深度压缩图片 |
| gulp-concat  | 合并文件 |
| gulp-changed  | 检查改变状态 |
| browser-sync  | 浏览器实时刷新 |
